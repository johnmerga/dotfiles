## :checkered_flag: Starting

```bash

# Clone this project to home direcotry
$ git clone https://github.com/johnmerga/dotfiles ~/dotfiles

# Go to the project folder
$ cd ~/dotfiles

# give permission to the install.sh file
$ chmod +x install

# Run the install file
$ ./install

# Enjoy


```
